C'est quoi JavaScript?

JavaScript est un langage dynamique, interprété and faiblement typé langage de programmation qui a été construit dans le but de fonctionner dans les navigateurs

Cela fait partie de pratiquement toutes les pages que vous visitez sur le Web, car c'est la seule "chose" dans le développement Web qui vous permet (= le développeur) de modifier quelque chose sur la page au moment de l'exécution. Fondamentalement, chaque fois que quelque chose s'ouvre, se ferme, se déplace ou change sur une page Web, JavaScript sera impliqué.

Il y a aussi beaucoup de pages Web qui sont presque entièrement alimentées par JavaScript (c'est-à-dire où une grande partie du travail de rendu se produit dans le navigateur) - ces pages utilisent généralement des frameworks comme Angular.

Qu'allez-vous apprendre dans cette formation ?

Ce cours enseigne JavaScript à partir de zéro - aucune connaissance préalable de JavaScript n'est requise.

Nous plongerons dans toutes les bases et les concepts de base ainsi que dans les fonctionnalités et modèles avancés jusqu'aux concepts et sujets d'experts.

Prérequis du formation

  • Connaissances de base en HTML et CSS est requis
  • Vous N'AVEZ PAS besoin d'être un développeur Web avancé ou expert
  • AUCUNE connaissance JavaScript n'est requise.
